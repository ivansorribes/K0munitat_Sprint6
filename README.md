## 🚀 Getting started 

Run the following commands in the root folder (K0MUNITAT_SPRINT4):

```
composer install
```

```
./vendor/bin/sail up --build
```

👩‍💻In another terminal:

```
npm install
```

```
npm run dev
```

## 🛰 Run the app

Now the app should be ready in the following URL:

```
http://localhost
```
